# Design References

优秀网页/应用界面设计参考收集库。

## 结构

```
design-references/
├── references/      # 收集的页面源文件（HTML/CSS）
├── notes/           # 分析笔记、设计模式总结
└── README.md
```

## 原则

- 只收藏**值得学习**的设计，宁缺毋滥
- 每一条都附分析笔记，说明为什么好、好在哪、怎么借鉴
- 尽量保存完整 HTML 源文件，避免链接失效
- 以中文记录，面向中文设计社区
